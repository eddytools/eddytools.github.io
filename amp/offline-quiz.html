Offline Assessment System

Features of Offline Assessment

Use Case: Consider there are around 7000 teachers in rural areas where there is no or limited connectivity who need to take this offline test. Now lets suppose there will be some 700 laptop devices/mobile devices provided to them and hence each device will be used by around 10 or more teachers to take the test by logging in with their username and password. Before we send the devices to these rural areas we will login as admin and sync the system with internet to download the tests. Now what we need is:

1. After syncing with the internet for the first time with any user login; test should be available offline when other teachers login using the same laptop offline.

2. Multiple attempts should not be allowed in some cases; admin must have the rights to restrict the number of attempts for each test.

3. All the test results must be synced with our online system and grade books linked with all user profiles should be updated with the results.

Our Solution

Eddytools Offline Assessment system can work on offline mode(without internet) and can sync back results once main server is connected to internet.
